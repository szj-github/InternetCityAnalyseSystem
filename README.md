# 互联网城市分析系统

## 摘要

本项目将实现对全国（中国）各大城市的数据进行采集，采集内容包括招聘情况，薪资 职位 待遇情况，以及每个城市的天气房价等等，数据可视化将以动态图表的展示在Web端，并且提供职位搜索匹配功能。
